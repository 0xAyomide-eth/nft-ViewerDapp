# nft-ViewerDapp

A decentralized application that shows users the nfts they own ethereum

## 🛠️ Tech Stack
*   **Frontend:** HTML / CSS / Javascript
*   **Blockchain Communication:** web3js &  Alchemy
*   **Network:** Ethereum 

## HOW TO SETUP 
1. clone the repository
2. make sure you have node js installed, then npm install
3. register on alchemy.com and get an APIKEY needed for this project
4. create a .env file and input your alchemy APIKEY (VITE_ALCHEMY_KEY=apikey)
5. make sure you have metamask installed or any EVM compatabile wallets
6. then run "npx vite"
